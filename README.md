# DocBook — نظام إدارة المواعيد الطبية 🏥

<div dir="rtl">

نظام إدارة مواعيد ومرضى للأطباء — **تطبيق ويب تقدمي (PWA)** يعمل على جميع الأجهزة بدون إنترنت.

## ✨ المميزات

- 📅 **إدارة المواعيد** — جدولة وتتبع مواعيد المرضى
- 👥 **ملفات المرضى** — سجلات طبية كاملة لكل مريض
- ⏱️ **عداد الزيارة** — تتبع مدة كل زيارة تلقائياً
- 📊 **إحصائيات** — تقارير وإحصاءات الكثافة والمرضى
- 🔒 **تسجيل دخول آمن** — نظام مصادقة بسيط
- 📱 **PWA** — يُثبَّت كتطبيق على الهاتف والجهاز اللوحي
- 🌐 **يعمل بدون إنترنت** — يحفظ البيانات محلياً
- 🌙 **تصميم RTL** — دعم كامل للعربية

## 🚀 رفع على GitHub Pages

### الخطوة 1: إنشاء Repository

```bash
git init
git add .
git commit -m "feat: DocBook PWA v1.0.0"
git branch -M main
git remote add origin https://github.com/اسمك/docbook.git
git push -u origin main
```

### الخطوة 2: تفعيل GitHub Pages

1. اذهب إلى **Settings** في الـ repository
2. اختر **Pages** من القائمة الجانبية
3. تحت **Source** اختر: `Deploy from a branch`
4. اختر **Branch**: `main` — **Folder**: `/ (root)`
5. اضغط **Save**

### الخطوة 3: الوصول للتطبيق

بعد دقيقتين يكون التطبيق متاحاً على:
```
https://اسمك.github.io/docbook/
```

## 📁 هيكل الملفات

```
docbook/
├── index.html          ← التطبيق الرئيسي (كل شيء)
├── manifest.json       ← إعدادات PWA (الاسم، الأيقونة، الألوان)

├── icons/
│   ├── icon-192.png    ← أيقونة صغيرة (للتثبيت)
│   └── icon-512.png    ← أيقونة كبيرة (Splash Screen)
├── screenshots/        ← صور التطبيق (اختياري)
└── README.md           ← هذا الملف
```

## 📱 تثبيت التطبيق (PWA)

### على الهاتف (iOS):
1. افتح الرابط في **Safari**
2. اضغط **Share** ثم **Add to Home Screen**

### على الهاتف (Android):
1. افتح الرابط في **Chrome**
2. اضغط **Install App** أو ستظهر نافذة تلقائية

### على الحاسوب:
1. افتح الرابط في **Chrome** أو **Edge**
2. اضغط أيقونة التثبيت في شريط العنوان ＋

## ⚙️ تعديل إعدادات PWA

### تغيير اسم التطبيق — `manifest.json`:
```json
{
  "name": "اسم تطبيقك",
  "short_name": "اسم قصير"
}
```

### تغيير لون الشريط العلوي — `manifest.json`:
```json
{
  "theme_color": "#0d9488"
}
```

## 🔧 تشغيل محلياً

```bash
# خيار 1 — Python
python3 -m http.server 8080

# خيار 2 — Node.js
npx serve .

# ثم افتح: http://localhost:8080
```

> ⚠️ لا تفتح `index.html` مباشرة بـ `file://` — يجب خادم HTTP لتعمل ميزة PWA.

## 📊 البيانات

جميع البيانات تُحفظ في **localStorage** بالمتصفح — لا يوجد قاعدة بيانات خارجية.

| المفتاح | المحتوى |
|---------|---------|
| `docbook_session` | بيانات الجلسة |
| `docbook_records` | المواعيد |
| `docbook_patients` | المرضى |
| `docbook_timer_stats` | إحصائيات العداد |
